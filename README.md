# 🎬 What Should I Watch Tonight?

AI-powered movie, series, and music recommender based on your mood.

## 🚀 Live Demo
👉 https://your-vercel-link

## 🧠 How It Works
- User enters mood
- Frontend sends request to backend
- Backend calls Groq API (LLaMA 3)
- Returns 3 recommendations with reasons

## 🛠️ Tech Stack
- HTML, CSS, JavaScript
- Node.js + Express
- Groq API (LLaMA 3.3)
- Vercel + Render

## 🔐 Security
- API key stored in .env
- Never exposed in frontend

## 🎯 Why I Built This
I wanted to build a real AI-powered app from scratch without tutorials, focusing on understanding every line of code.

## 📌 Features
- Mood-based recommendations
- Clean Netflix-style UI
- Loading spinner + error handling
